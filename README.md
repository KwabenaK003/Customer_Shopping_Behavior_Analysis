# Customer_Shopping_Behavior_Analysis
Customer shopping behavior Analysis

## Overview

This project demonstrates a complete data analytics workflow from raw dataset ingestion to executive presentation. The process includes data loading and cleaning in Python, exploratory data analysis (EDA), SQL querying in PostgreSQL, dashboard development in Power BI, and final reporting through a structured presentation.

The objective was to transform raw data into actionable insights and present them in a clear, business-ready format.

## Dataset

* Structured dataset (CSV format)
* Contains transactional/business-related records
* Includes categorical, numerical, and date/time features
* Used for exploratory analysis, SQL querying, and dashboard visualization

## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib/Seaborn)
* **PostgreSQL** (SQL querying and data validation)
* **Power BI** (Interactive dashboard development)
* **Gamma** (Professional presentation creation)
* **VS Code / pgAdmin** (Development and database management)

## Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas
* Inspected structure, data types, and summary statistics

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions, trends, and correlations
* Identified missing values and inconsistencies
* Generated visual summaries to understand patterns

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized formats (dates, categories)
* Validated data integrity

### 4. SQL Analysis (PostgreSQL)

* Loaded cleaned data into PostgreSQL
* Executed analytical queries:

  * Aggregations
  * Grouped summaries
  * Trend analysis
  * Filtering and joins
* Verified business metrics using SQL

### 5. Power BI Dashboard

* Connected Power BI to PostgreSQL
* Designed interactive visualizations
* Built KPIs and summary metrics
* Created drill-down and filtering functionality

### 6. Reporting & Presentation

* Compiled findings into a structured report
* Built executive-level presentation using Gamma
* Highlighted key insights, trends, and recommendations

## Dashboard

The Power BI dashboard includes:

* Key performance indicators (KPIs)
* Trend analysis over time
* Category-based performance breakdown
* Interactive filters and slicers
* Clean, business-focused layout

Designed for stakeholder-level decision-making.


## Results & Insights

* Identified major performance drivers
* Detected trends and seasonal patterns
* Highlighted underperforming segments
* Provided data-backed recommendations for improvement

The project demonstrates the ability to translate raw data into strategic insights.
